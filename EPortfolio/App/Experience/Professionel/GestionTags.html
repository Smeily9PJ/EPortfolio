2014 - Gestion des Tags

Environnement
Java, JQuery

Sujet

Ce projet fût réalisé lors d'un stage de 3 mois à l'IRIT (Institut de Recherche Informatique de Toulouse). Le but étant de créer une application permettant de recensser des tags à intervalle régulier.

Déroulement

Première partie : développement de l'application java de recenssement de tag

Deuxieme partie : ajout de fonctionnalité sur un plugin chrome
