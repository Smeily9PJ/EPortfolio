Langage CSS et SASS

Dans mon expérience

J'ai découvert le SASS lors de mon arrivée sur paris. Connaissant déjà le langage CSS depuis le début de mes études post-BAC, j'ai rapidement pû apprendre ce langage qui simplifie grandement l'utilisation du CSS.

Aussi bien dans le domaine étudiant que dans le domaine professionel, quand je devais utiliser ce langage dans un projet, je créé (ou utilisait s'il existait déjà) un fichier général qui contient toutes les variables (pour la SASS) et toutes les propriétés générales et un fichier par page afin que pouvoir plus facilement lire le code.

Regard critique

Le plus dur dans ce langage est de le rendre clair et d'éviter la répétition, surout lorsqu'on est plusieur à coder.

Pour éviter au maximum la répétition, il faut communiquer avec l'équipe pour indiquer ou demander si ce qu'on cherche existe ou s'il faut le créer.

Evolution

Je compte continuer à travailler sur la clairté du code et sur les éléments pouvant aider à simplifier le langage.
